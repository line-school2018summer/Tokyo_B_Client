# /register [get / post]
## GET
```
{"error": 0,
"content": {
    "message": "/register[get]"
    }
}
```
### request
```
{"target": "/register",
"authenticated": {authenticated},
"id": {id},
"name": {name},
"password": {password},
"password_confirm": {password}}
```
### response
* エラー有りの場合
    * すでにログインしている状態での/registerへのpostは認められません。
    * 既存アカウントに重複するような"id"は使用できません。
    * 3文字以下、13文字以上、英数字以外の"id"は認められません。
    * 0文字、もしくは32文字以上の"name"は認められません。
    * 5文字以下、13文字以上、英数字以外の"password"は認められません。
    * "password"と"password_confirm"は等しくなければいけません。
```
{"error": 1,
"content": {
    "authenticated": {0 or 1},
    "exist_id": {0 or 1},
    "bad_id": {0 or 1},
    "bad_name": {0 or 1},
    "bad_password": {0 or 1},
    "password_confirm_does_not_match": {0 or 1}
    }
}
```
* エラーなしの場合
```
{"error": 0,
"content": {
    "logged_id": {id},
    "logged_pass" {password}
    }
}
```

# /login [get / post]
## GET
```
{"error": 0,
"content": {
    "message": "/login[get]"
    }
}
```
### request
```
{"target": "/login", 
"authenticated": {authenticated},
"id": {id},
"password": {password}
```
### response
* エラー有りの場合 
    * すでにログインしている状態での/registerへのpostは認められません。
    * 存在している"id"でなければいけません。
    * "id"が存在していた場合、"password"が合致していなければいけません。
    * (unexist_id が 1 の時、invalid_password は自動的に 0 になるので、両者が常に同じ結果になることはありえません。)
```
{"error": 1, 
"content": {
    "authenticated": {0 or 1},
    "unexist_id": {0 or 1},
    "invalid_password": {0 or 1}
    }
}
```
* エラーなしの場合
```
{"error": 0, 
"content": {
    "logged_id": {id},
    "logged_pass" {password}
    }
}
```

# /account_modify [get / post]
## GET
```
{"error": 0,
"content": {
    "message": "/account_modify[get]"
    }
}
```
### request
"modify"内は変更先の情報を入力してください。空白は変更なしと捉えます。
```
{"target": "/account_modify",
"authenticated": {authenticated},
"id": {id},
"password": {password},
"modify": {
    "id": {modify_id},
    "name": {modify_name},
    "password": {modify_password},
    "password_confirm": {modify_password}},
}
```
### response
* エラー有りの場合
    * ログインしていない状態での/account_modifyへのpostは認められません。
    * 存在しない"id", "password"でのpostは認められません。
    * 既存の他のアカウントに重複するような"modify"-"id"は使用できません。
    * 3文字以下、13文字以上、英数字以外の"modify"-"id"は認められません。
    * 0文字、もしくは32文字以上の"modify"-"name"は認められません。
    * 5文字以下、13文字以上、英数字以外の"modift"-"password"は認められません。
    * "modify"-"password"と"modify"-"password_confirm"は等しくなければいけません。
```
{"error": 1,
"content": {
    "not_authenticated": {0 or 1},
    "exist_id": {0 or 1},
    "bad_id": {0 or 1},
    "bad_name": {0 or 1},
    "bad_password": {0 or 1},
    "password_confirm_does_not_match": {0 or 1}
    }
}
```
* エラーなしの場合
```
{"error": 0,
"content": {
    "new_id": {id},
    "new_name": {name},
    }
}
```
