### JSONの仕様
~~~
# /register
## GET
｀｀｀{"error": 0,
"content": {
    "message": "/register[get]"
    }
}｀｀｀
### request
｀｀｀{"target": "/register", 
"id": {id},
"name": {name},
"password": {password}}｀｀｀
### response
* エラー有りの場合
｀｀｀{"error": 1, 
"content": {
    "exist_id": {0 or 1}, 
    "bad_id": {0 or 1}, 
    "bad_password": {0 or 1}, 
    "password_confirm_does_not_match": {0 or 1} 
    }
}｀｀｀
* エラーなしの場合
｀｀｀{"error": 0, 
"content": {
    "logged_id": {id}
    }
}｀｀｀
~~~
